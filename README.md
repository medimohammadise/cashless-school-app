
# Project Title

Cashless School


## Installation and running
### Android

Install packages with yarn:

```bash
  yarn install
```

Running:
If you have an emulator on your machine:

```bash
  yarn android 
```
or 
```bash
  npx react-native run-android
```
If not, connect your device via USB and

```bash
  yarn android 
```
or 
```bash
  npx react-native run-android
```
